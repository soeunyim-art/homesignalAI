# 홈시그널AI (HomeSignal AI) 
UX/UI 와이어프레임 상세 설계서

본 문서는 레퍼런스 이미지의 UI 요소를 기반으로 홈시그널AI의 화면 구조, 컴포넌트 속성, 그리고 데이터 시각화를 위한 정밀 컬러 시스템을 정의합니다.

## 1. 글로벌 네비게이션 및 헤더 (GNB)

- **Brand Logo:** 3x3 민트 도트 그리드 아이콘 + "홈시그널AI" (Pretendard Bold).
- **Search Bar:** 배경 `#111827`, 보더 `#1F2937`, 텍스트 "분석할 지역 또는 아파트명을 입력하세요".
- **Utility:** 지역 선택 셀렉터, 알림 아이콘, 프로필/로그인 버튼.

## 2. 마켓 KPI 카드 (Top Metrics)

화면 상단에 배치되어 시장의 거시적 흐름을 실시간으로 노출합니다.

- **카드 구조:** 아이콘 + 지표명 + 현재 수치 + 등락폭(Sparkline).
- **항목:** 서울 아파트 평균(12.4억), 전국 매매지수(102.8), 수도권 거래량(8,420건), 전세가율(68.2%).
- **컬러 규칙:** 상승 시 `Mint(#4ADE80)`, 하락 시 `Red(#EF4444)`.

## 3. 메인 분석 영역: 지도 및 공급 레이어 (Map & Supply)

### [Interactive Map]

- **Base Style:** Dark-mode 벡터 지도.
- **공급 히트맵 범례 (Supply Heatmap Legend):**
    1. **적정 (Stable):** `#14B8A6` (Teal) - 공급 수급이 안정적인 상태.
    2. **보통 (Moderate):** `#F59E0B` (Amber) - 향후 공급 주의 필요.
    3. **과다 (High):** `#EA580C` (Orange) - 단기 공급 집중 구역.
    4. **과잉 경고 (Critical):** `#991B1B` (Deep Red) - 대규모 입주 폭탄 리스크.

### [공급 예정 물량 위젯 (Right Side)]

- **상태 뱃지:** `주의 단계` (Amber 배경에 검정 텍스트).
- **타임라인 바:** 연도별 물량을 가로 막대 차트로 시각화.
    - 2026 상반기: 2,100세대 (주의 - Orange)
    - 2026 하반기: 1,800세대 (보통 - Amber)
    - 2027 상반기: 900세대 (적정 - Teal)

## 4. AI 시뮬레이션 영역 (Analysis & Scenario)

### [AI 가격 예측 차트]

- **실거래선 (Actual):** `#FFFFFF` (White) 1.5px 실선.
- **예측선 (Predicted):** `#3B82F6` (Electric Blue) 점선.
- **확률 밴드 (Probability Band):** `#3B82F6` 배경, 투명도 10~20%의 Area Chart.
- **시나리오 카드:**
    - **비관:** 가격 강조 컬러 `#EF4444` (Red).
    - **기본:** 가격 강조 컬러 `#3B82F6` (Blue).
    - **낙관:** 가격 강조 컬러 `#4ADE80` (Mint).

## 5. 실시간 뉴스 시그널 (News Sentiment)

AI가 뉴스를 분석하여 시장 심리를 수치화합니다.

- **뉴스 카드 구성:** 카테고리 태그, 매체명, 경과 시간, 제목, 요약, 감성 뱃지.
- **감성 시그널 컬러 규정:**
    - **긍정 시그널 (Positive):** * 우상향 아이콘 및 퍼센트: `#4ADE80` (Mint)
        - "긍정 시그널" 배지: 배경 `rgba(74, 222, 128, 0.1)`, 텍스트 `#4ADE80`.
    - **부정 시그널 (Negative):**
        - 우하향 아이콘 및 퍼센트: `#FB7185` (Soft Red)
        - "부정 시그널" 배지: 배경 `rgba(251, 113, 133, 0.1)`, 텍스트 `#FB7185`.

## 6. 핵심 디자인 시스템 (UI/UX Rules)

| 구분 | Hex Code | 사용 위치 |
| --- | --- | --- |
| **Main BG** | `#0B1220` | 전체 배경 (Deep Navy) |
| **Surface** | `#111827` | 모든 카드 및 대시보드 모듈 |
| **Primary** | `#4ADE80` | 홈시그널AI 브랜드 컬러, 긍정/상승 |

| **Danger** | `#EF4444` | 과잉 경고, 부정 뉴스, 비관 시나리오 |
| **Grid Dot** | `#FFFFFF` | 배경에 5~8% 투명도로 배치되는 3x3 모티프 패턴 |
