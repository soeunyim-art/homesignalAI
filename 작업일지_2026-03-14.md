## 1. 예측 모델 개선 (predict_model.py)

### 1-1. 예측값 클리핑 (CLIP_BOUNDS)
- 거래량이 적은 동에서 비현실적 예측 발생 (길음동 -30%, 장위동 -38%)
- 원인: 데이터 부족으로 인한 Ridge 모델 과적합
- 해결: 예측 변동률 상하한 클리핑 적용

```python
CLIP_BOUNDS = {1: 10.0, 2: 15.0, 3: 20.0}  # 1개월 ±10%, 2개월 ±15%, 3개월 ±20%
```

### 1-2. 신뢰도 점수 (confidence_score)
- 월 거래 건수 기반으로 신뢰도 0~100 산출
- 거래량 20건 이상 → 신뢰도 100, 적을수록 낮아짐

```python
def calc_confidence(trade_count: int) -> int:
    return min(100, max(0, int(trade_count * 5)))
```

### 1-3. Supabase 스키마 업데이트
```sql
ALTER TABLE predictions ADD COLUMN IF NOT EXISTS confidence_score INTEGER;
```
