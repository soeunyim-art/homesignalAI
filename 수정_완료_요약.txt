챗봇 실행 방법
==========================================

1. Next.js 서버 시작:
   npm run dev

2. 브라우저에서 접속:
   http://localhost:3000

3. 우측 하단 "홈시그널 AI에게 물어보기" 버튼 클릭

4. 질문 예시:
   - 동대문구 아파트 가격 전망은?
   - 최근 뉴스 요약해줘
   - 가장 상승률 높은 동은?

문제 해결
==========================================

Q: "API 키가 유효하지 않습니다" 오류
A: .env 파일의 ANTHROPIC_API_KEY 확인 후 서버 재시작

Q: 챗봇 응답이 느림
A: 정상입니다. RAG 시스템이 DB 조회하므로 2-3초 소요

Q: "데이터가 없습니다" 응답
A: python predict_model.py 실행하여 예측 데이터 생성
