## 챗봇 작동 원리

### RAG (Retrieval-Augmented Generation)

챗봇은 단순히 미리 학습된 지식으로만 답변하지 않습니다.
대신 **실시간으로 데이터베이스를 검색**하여 최신 정보를 제공합니다.

```
사용자 질문
   ↓
질문 의도 분석 (예측? 뉴스? 비교?)
   ↓
Supabase에서 관련 데이터 검색
   ↓
검색된 데이터로 컨텍스트 구성
   ↓
Claude AI에게 컨텍스트와 함께 질문 전달
   ↓
AI가 데이터 기반 답변 생성
   ↓
사용자에게 응답 표시
```

### 검색되는 데이터

1. **예측 데이터** (predictions 테이블)
   - 동별 평균 매매가 예측 (1/2/3개월 후)
   - 변동률 및 신뢰도 점수

2. **뉴스 시그널** (news_signals 테이블)
   - 최근 부동산 뉴스
   - 정책, 규제, 재개발 관련 소식

3. **실거래 통계** (apt_trade 테이블)
   - 2025년 이후 실거래 내역
   - 지역별 평균 거래가

## 기능 테스트

### 테스트 1: 전체 시스템 진단

```bash
python test_chatbot_detailed.py
```

이 스크립트는 다음을 확인합니다:
- ✅ 환경 변수 설정 (ANTHROPIC_API_KEY, SUPABASE 키)
- ✅ Supabase 데이터베이스 연결
- ✅ 데이터 검색 (예측, 뉴스, 거래 데이터)
- ✅ Claude API 연결

### 테스트 2: API 직접 테스트

```bash
# 터미널 1: Next.js 서버 시작
npm run dev

# 터미널 2: API 테스트 실행
node test_chatbot.js
```

이 스크립트는 3가지 질문을 자동으로 보내고 응답을 확인합니다.

## 문제 해결

### "API 키가 유효하지 않습니다" 오류

**원인**: `.env` 파일에 ANTHROPIC_API_KEY가 없거나 잘못됨

**해결**:
1. `.env` 파일을 열고 다음 줄이 있는지 확인:
   ```
   ANTHROPIC_API_KEY=sk-ant-api03-...
   ```
2. 키가 올바른지 확인
3. Next.js 서버 재시작 (`npm run dev`)

### "데이터를 가져올 수 없습니다" 오류

**원인**: Supabase 연결 문제

**해결**:
```bash
# Supabase 연결 테스트
python test_supabase_connection.py

# 전체 진단
python test_chatbot_detailed.py
```

### 챗봇이 "데이터가 없습니다"라고 답변

**원인**: 해당 지역의 예측 데이터가 없음

**해결**:
```bash
# ML 모델 실행하여 예측 데이터 생성
python predict_model.py
```

### 챗봇 응답이 느림

**정상**: RAG 시스템이 여러 데이터베이스 쿼리를 실행하므로 2-3초 소요됩니다.

**개선 방법**:
- Supabase 인덱스 추가
- 자주 묻는 질문 캐싱
- 스트리밍 응답 구현 (향후 업데이트)

### "model not found" 오류

**원인**: 잘못된 Claude 모델 이름 (이미 수정됨)

**확인**: `app/api/chat/route.ts` 파일에서 모델 이름이 `claude-sonnet-4-5-20250929`인지 확인

### 예시 1: 가격 예측 질문

**질문**: "동대문구 아파트 가격 전망은?"

**챗봇 응답**:
```
동대문구 아파트 시장 분석 (2026-03-20 기준)

📊 주요 지표:
- 분석 동 수: 15개
- 평균 매매가: 약 8.5억원
- 1개월 후 평균 변동률: +1.2%

상승 예상 지역 TOP 3:
1. 휘경동: 8.2억 → 8.4억 (+2.3%)
2. 답십리동: 9.1억 → 9.3억 (+1.8%)
3. 장안동: 7.8억 → 7.9억 (+1.5%)

💡 투자 포인트:
- 전반적으로 완만한 상승세 예상
- 재개발 예정 지역 (답십리, 장안) 주목
- 지하철역 접근성 좋은 단지 선호

※ 본 정보는 투자 참고 자료이며, 실제 투자 결정은 신중히 하시기 바랍니다.
```

### 예시 2: 뉴스 질문

**질문**: "최근 뉴스 요약해줘"

**챗봇 응답**:
```
최근 부동산 뉴스 요약 (2026-03월)

📰 주요 뉴스:
1. "강북·노원 '전세 증발'의 진실" (3월 20일)
   - 대규모 단지에서 전세 물량 급감
   - 월세 전환 가속화

2. "동대문구 답십리 재개발 본격화" (3월 15일)
   - 총 4,000세대 규모
   - 2027년 착공 예정

3. "기준금리 동결, 부동산 시장 안정화 전망" (3월 10일)
   - 한국은행 금리 3.25% 유지
   - 시장 관망세 지속 예상

📈 시장 영향:
- 전세→월세 전환으로 매매 시장 영향 제한적
- 재개발 지역 관심도 상승
- 금리 안정으로 거래량 회복 기대

※ 뉴스 요약은 참고용이며, 투자 판단 시 추가 조사를 권장합니다.
```

## 사용 팁

### 1. 구체적으로 질문하세요

❌ 나쁜 예: "집값 어때?"
✅ 좋은 예: "동대문구 휘경동 아파트 1개월 후 가격 전망은?"

### 2. 지역명을 정확히 입력하세요

지원 지역: 동대문구, 성북구, 중랑구, 강북구, 도봉구
동 단위: "휘경동", "장안동", "답십리동" 등

### 3. 여러 질문을 나눠서 하세요

❌ 나쁜 예: "동대문구 가격이랑 뉴스랑 거래량이랑 전세가율 다 알려줘"
✅ 좋은 예:
   - 첫 번째 질문: "동대문구 가격 전망은?"
   - 두 번째 질문: "동대문구 관련 뉴스는?"

### 4. 제안 질문을 활용하세요

챗봇 창 하단에 표시되는 4개의 제안 질문 버튼을 클릭하면 자동으로 질문이 입력됩니다.
