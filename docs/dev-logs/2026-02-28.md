## 🚧 삽질 기록 (Troubleshooting)

### 문제점
- supabase Python 패키지 import 시 `ModuleNotFoundError: No module named 'pyiceberg'` 발생
- storage3 모듈이 pyiceberg를 analytics용으로 의존하고 있어, pyiceberg 미설치 환경에서 Supabase 클라이언트가 Mock으로 fallback됨

### 해결 방법
- `pyproject.toml`에 `pyiceberg>=0.5.0` 의존성 추가
- Windows 환경에서는 pyiceberg 설치가 완료되지 않을 수 있어, `pip install pyiceberg` 수동 실행 또는 uv sync로 해결
