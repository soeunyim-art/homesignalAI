# 홈시그널 AI 챗봇 - 빠른 시작 가이드

## 30초 시작하기

```bash
# 1. 서버 시작
npm run dev

# 2. 브라우저에서 접속
# http://localhost:3000

# 3. 우측 하단 버튼 클릭
# "홈시그널 AI에게 물어보기"
```

## 테스트 명령어

### 전체 시스템 체크 (권장)
```bash
python test_chatbot_detailed.py
```

출력 예시:
```
✅ 환경 변수          : ✅ 통과
✅ Supabase 연결     : ✅ 통과
✅ 데이터 검색        : ✅ 통과
✅ Anthropic API     : ✅ 통과
```

### API만 테스트
```bash
# 터미널 1
npm run dev

# 터미널 2
node test_chatbot.js
```

## 문제 해결

### "API 키가 유효하지 않습니다"
```bash
# .env 파일 확인
notepad .env

# 다음 줄이 있는지 확인:
# ANTHROPIC_API_KEY=sk-ant-api03-...

# 서버 재시작
# Ctrl+C로 종료 후
npm run dev
```

### "데이터가 없습니다"
```bash
# 예측 데이터 생성
python predict_model.py

# 서버 재시작
npm run dev
```

### 챗봇이 안 열려요
```bash
# 브라우저 콘솔 확인 (F12)
# 오류 메시지 확인

# 또는 전체 진단 실행
python test_chatbot_detailed.py
```
